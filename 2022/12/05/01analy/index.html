读入文件的主要函数.

while ((ch = fgetc(fh)) != EOF) string_builder_append(sb, ch);

以上这一句是分析每一个char的语句。

int fgetc(FILE *stream)
{
    char buf[1];
    int res = read(stream->fd, buf, 1);
    if (res <= 0) return EOF;
    return buf[0] & 0xff;
}

在fgetc函数里面，通过read函数读入每一个字符，返回给buf[0]，然后取第八位传给while循环。

然后进入string_builder_append(sb, ch); 函数。

char string_builder_append(StringBuilder *sb, char ch)
{
    vector_push_back(sb->data, (void *)ch);
    return ch;
}

该函数主要操主调函数传下来的 StringBuilder *sb。

其中 sb->data里的size表示字符数。

返回 string_builder_get(sb); 该函数。

char *string_builder_get(StringBuilder *sb)
{
    int size = vector_size(sb->data);
    char *ret = safe_malloc(size + 1);
    for (int i = 0; i < size; i++) ret[i] = (char)vector_get(sb->data, i);
    ret[size] = '\0';
    return ret;
}

该函数将采集到的所有字符放入 char ret[]数组中并返回主调函数。

  1. 先后两个 read_asm_param()分别分析左右操作数。

  2. 接着用 vector_push_back(code, new_binop_code((int)kv_value(kv), lhs, rhs));将其整合到一起。即将信息填充给第一个参数 code。我们从void vector_push_back(Vector *vec, void *item)的源码就可以看到填充过程。

  3. void vector_push_back(Vector *vec, void *item)
{
    if (vec->data == NULL || vec->size == vec->rsved_size) {
        vec->rsved_size *= 2;
        void **ndata = (void **)safe_malloc(sizeof(void *) * vec->rsved_size);
        memcpy(ndata, vec->data, vec->size * sizeof(void *));
        vec->data = ndata;
    }

    vec->data[vec->size++] = item;
}
//这里的 vec参数就是传进来的code的地址.

read_asm_param();

该函数总结来说就是分析一个汇编语句的其中一个操作数的内容。==例如：mov %rax, %rdi==的 %rax。

​ 看看这个函数的返回值

数据结构

Vector

该项目均采用vector数据结构

struct Vector {
    int size, rsved_size;
    void **data;
};

KeyValue

key存字符串

value存对应的值，用于匹配查询

struct KeyValue {
    const char *key;
    void *value;
};

Map

struct Map {
    Vector *data;
};

StringBuilder

struct StringBuilder {
    Vector *data;
};

Source

src用于存储输入的汇编文件

line和cloumn表示正在处理的行列

typedef struct {
    int line, column;
