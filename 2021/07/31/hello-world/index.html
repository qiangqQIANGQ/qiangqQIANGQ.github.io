萌新电脑操作技巧

Created2021-07-31Updated2021-08-02

    Welcome! 对于我们专业的同学们来说电脑是经常接触到的，就正如我们的桌面一样，保持电脑的干净整洁，不仅能让心情愉快，还能提高工作效率。

    我家那台十年前的台式机，如今里面充斥着各种弹窗广告，c盘被挤满，好几个杀毒软件养蛊，浏览器被篡改，这些都使得这台电脑速度慢，频繁卡顿，安全性低，不整洁。或许你并不希望在敲码或追剧正尽兴时忽然冒出一个砍一刀999级的广告，因此根据我的一些经历在这里写了一些关于保养电脑的想法。持续更新。建议使用电脑或pad浏览，这样有侧边栏大纲哦。

    当然知识受限，如果有错误或不完备的还请你多多指教，谢谢啦。

快捷按键

经常使用电脑的同学有必要知道某些使用频率高的快捷键，这有帮助于提高操作效率。

文档编辑类

  • CTRL + C/V 复制(Copy)/粘贴(Paste)
  • CTRL + A 全选(All)
  • CTRL + Z 撤销(Unmake)
  • CTRL + X 剪贴
  • CTRL + S 保存(Save)
  • CTRL + Y 反撤销
  • win + V 打开剪贴板
  • CTRL + 鼠标滚轮 网页等放大缩小
  • CTRL + ALT + A （QQ登录时）截屏
  • CTRL + N　新建（New）

电脑操作类

  • CTRL + SHIFT + ESC　打开任务管理器
       ps.尤其是电脑出现软件卡死时打开任务管理器，右键单击该软件可以强制退出；当电脑莫名卡顿时也可以通过查看CPU占用比异常高的后台程序通过上网搜索寻求解放方案。（但后台和进程程序最好不要擅自强制关闭）
  • 触摸板三指左右滑 切换任务界面
  • 触摸板三指上下滑 返回页面/返回桌面

维护技巧

关于软件安装

  1. 下载地点：最好选择官网，(一定看清楚哦，很多时候百度出来的位置靠前一两个的反而不是官网)或者正规软件商店，不要选择百度里的非官方渠道，避免不兼容或捆绑病毒。
  2. 下载方式：先选择<另存为>到桌面，下载完安装包后桌面点击运行，等软件安装完毕后直接将桌面上的安装包拖进回收站。
  3. 点开安装包后，不要一直点击下一步，选择<自定义安装>而不是<快速安装>！因为如果有捆绑软件此时需要叉掉，并且不要安装到默认c盘，此时建议在D盘新建一个文件夹，然后将该软件安装到这个新建的文件夹里。

关于软件卸载

  • 卸载我推荐一个软件叫做 Geek Uninstaller 官网链接,因为大多卸载方式存在文件残留以及注册表信息存留，此款软件无广告，轻量而且操作方便，推荐安装。

关于杀毒软件

  • 事实上windows电脑自带Windows defender能够有防护效果，所以通常来讲电脑并不需要额外安装杀毒软件。如果你的电脑还有360，腾讯电脑管家等等，建议你卸载，这些软件弹窗广告多，占后台，篡改主页等等，有点流氓。如果实在想安装杀毒软件，火绒安全很不错。

关于文件分类

  • 电脑上一切软件，包括操作系统都是由那几个盘里的文件构成的，因此学会整理文件不仅自己感到整齐，也可以方便我们查找，不增加混乱度。比如前面的下载软件单独到一个文件夹，还比如安装包下载到桌面用了就删掉，而不是让电脑默认下到C盘的某个不知名的角落占用内存等等。

关于浏览器

  • 谷歌，火狐等浏览器都是不错的选择，但如果你不想下载，事实上win10自带的edge浏览器就是特别特别棒的选择！请一定不要单独下载什么360浏览器，2345浏览器等等。如果自己的主页被篡改了可以再网上搜索一下如何恢复edge默认主页。
  • edge浏览器占用少，速度快，并且干净整洁！！你可以对比一下edge主页和2345主页，很明显清爽很多嘛！！！
  • edge浏览器还有一些好用的功能可以自己去探索一下哦~

Tips:如果你的电脑已经充满了这些杀毒软件，不好的浏览器，就从上面那一款卸载软件Geek Uninstaller的安装开始吧，然后卸载掉这些东西~

关于内存占用

  • 净化开机启动项

    个别软件安装后默认加到开机启动里边，不仅拖慢开机速度，而且开机后默认启动到后台占用内存。
    打开任务管理器（CTRL+SHIFT+ESC）点击<启动>选项，禁用某些无关应用自启动，例如百度网盘等等。

软件推荐

    在这里推荐一些很好用的基础软件，如果你也有的话欢迎反馈，我会标注好推荐来源~

浏览器

浏览器是我们使用频率最高的软件，有很多扩展功能值得去探索，也能大大提高效率，并且变得好看。

  • Microsoft Edge浏览器

    windows 10自带的浏览器，干净好用，推荐使用，不过请注意一定不要更换主页为360什么的，使用默认主页。

  • Google chrome浏览器 官网链接

    扩展性强，市场排名第一浏览器，调试功能强大

  • Firefox浏览器 官网链接

    开源自由，扩展多，特别好

  • Safari浏览器 官网链接

    苹果系统自带，当然Windows用户也可以使用。

杀毒软件

  • 无
  • 如果确实很想安装一个，那就要 火绒安全 官网链接 吧~

清理软件

  • Ccleaner 官网链接

    如果卸载掉360安全卫士后你觉得没有清理软件了，那么这一款就是评价很高的清理软件，不过我没怎么用过，好像褒贬不一。

卸载软件

  • Geek Uninstaller 官网链接

    前面介绍过咯，直接安装就好啦，很棒。

压缩软件

  • 360zip 官网链接

    电脑里必备一款压缩软件，以后需要用到，虽然很多其他的360软件有点难用，但360zip确实很好，无广告，轻量好用。

  • 7-zip 或者 bandzip

    也有推荐这两款的，不过我的360zip的确很好用，所以没下过这两个，但是口碑也还不错。

播放器

  • PotPlayer 官网链接

    如果要播放本地视频，请不要选择爱奇艺播放器，这款软件值得，功能强大，界面简洁清爽，无广告，配置选项丰富~

检索软件

  • everything 官网链接

    一个字：超级强大！该款软件检索文件速度比Windows自带的检索速度快了不知多少倍！刚刚亲测了一下，打开<我的电脑>全局搜索 zip 关键词，搜了五分钟都没搜完qwq我就关了，而这个甚至不需要按Enter键直接秒出结果。并且支持正则表达式等高级搜索功能，强烈安利！

    Tips:下载x64位版本的哦，不要下错啦，可以选 <安装版64位>
