萌新电脑操作技巧

Created2021-07-31Updated2021-08-02

    Welcome to my blog.对于我们专业的同学们来说电脑是经常接触到的，就正如我们的桌面一样，保持电脑的干净整洁，不仅能让心情愉快，还能提高工作效率。

    我家那台十年前的台式机，如今里面充斥着各种弹窗广告，c盘被挤满，好几个杀毒软件养蛊，浏览器被篡改，这些都使得这台电脑速度慢，频繁卡顿，安全性低，不整洁。或许你并不希望在敲码或追剧正尽兴时忽然冒出一个砍一刀999级的广告，因此根据我的一些经历在这里写了一些关于保养电脑的想法。持续更新。建议使用电脑或pad浏览，这样有侧边栏大纲哦。

    当然知识受限，如果有错误或不完备的还请你多多指正，谢谢啦。

快捷按键

经常使用电脑的同学有必要知道某些使用频率高的快捷键，这有帮助于提高操作效率。

文档编辑类

  • CTRL + C/V 复制(Copy)/粘贴(Paste)
  • CTRL + A 全选(All)
  • CTRL + Z 撤销(Unmake)
  • CTRL + X 剪贴
  • CTRL + S 保存(Save)
  • CTRL + Y 反撤销
  • win + V 打开剪贴板
  • CTRL + 鼠标滚轮 网页等放大缩小
  • CTRL + ALT + A （QQ登录时）截屏
  • CTRL + N　新建（New）

电脑操作类

  • CTRL + SHIFT + ESC　打开任务管理器
       ps.尤其是电脑出现软件卡死时打开任务管理器，右键单击该软件可以强制退出；当电脑莫名卡顿时也可以通过查看CPU占用比异常高的后台程序通过上网搜索寻求解放方案。（但后台和进程程序最好不要擅自强制关闭）
  • 触摸板三指左右滑 切换任务界面
  • 触摸板三指上下滑 返回页面/返回桌面

维护技巧

关于软件安装

  1. 下载地点：最好选择官网，(一定看清楚哦，很多时候百度出来的位置靠前一两个的反而不是官网)或者正规软件商店，不要选择百度里的非官方渠道，避免不兼容或捆绑病毒。
  2. 下载方式：先选择<另存为>到桌面，下载完安装包后桌面点击运行，等软件安装完毕后直接将桌面上的安装包拖进回收站。
