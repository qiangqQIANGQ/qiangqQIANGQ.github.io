萌新电脑操作技巧

Created2021-07-31Updated2021-08-01

Welcome to ChaceQ’s little blog.对于我们专业的同学们来说，电脑应该算是经常接触到的，正如我们的桌面一样，保持电脑的干净整洁，不仅能让心情愉快，还能提高工作效率。比如我家那台十年前的台式机，如今里面充斥着各种弹窗广告，c盘被挤满，好几个杀毒软件养蛊，浏览器被篡改，这些都使得这台电脑速度慢，频繁卡顿，安全性低，不整洁。因此根据我的一些使用方法在这里写了一些关于保养电脑的板块。当然知识受限，如果有错误或不完备的还望指正，谢谢啦。

快捷按键

经常使用电脑的同学有必要知道某些使用频率高的快捷键，这有帮助于提高操作效率。

文档编辑类

  • CTRL + C/V 复制/粘贴
  • CTRL + A 全选
  • CTRL + Z 撤销
  • CTRL + X 剪贴
  • CTRL + S 保存
  • CTRL + Y 反撤销
  • win + V 打开剪贴板
  • SHIFT + 鼠标滚轮 图片等放大缩小
  • CTRL + ALT + A （QQ登录时）截屏

    电脑操作类

  • CTRL + SHIFT　+　ESC　打开任务管理器
       ps.尤其是电脑出现软件卡死时打开任务管理器，右键单击该软件可以强制退出；当电脑莫名卡顿时也可以通过查看CPU占用比异常高的后台程序通过上网搜索寻求解放方案。（但后台和进程程序最好不要擅自强制关闭）

保养技巧

关于安装软件

  1. 下载地点：最好选择官网，或者正规软件商店，不要选择百度里的非官方渠道，避免不兼容或捆绑病毒。
  2. 下载方式：先选择<另存为>到桌面，下载完安装包后桌面点击运行，等软件安装完毕后直接将桌面上的安装包拖进回收站。
  3. 点开安装包后，不要一股脑地点击下一步，选择<自定义安装>而不是<快速安装>！因为如果有捆绑软件此时需要叉掉，并且不要安装到默认c盘，此时建议在D盘新建一个文件夹，然后将该软件安装到这个新建的文件夹里。

关于卸载软件

  • 卸载软件的话我推荐一个软件叫做 Geek Uninstaller 官网链接,因为大多卸载方式存在文件残留以及注册表信息存留，此款软件无广告，轻量而且卸载软件操作方便，推荐安装。

关于杀毒软件

  • 事实上windows电脑自带Windows defender能够有防护效果，所以通常来讲电脑并不需要额外安装杀毒软件。如果你的电脑还有360，腾讯电脑管家等等，建议你卸载，这些软件弹窗广告多，篡改主页等等，有点流氓。如果实在想安装杀毒软件，火绒安全很不错。

文件分类

  • 众所周知，电脑上一切软件，包括操作系统都是由那几个盘里的文件构成的，因此学会整理文件不仅自己感到整齐，也可以方便我们查找，不增加混乱度。比如前面的下载软件单独到一个文件夹，还比如安装包下载到桌面用了就删掉，而不是让电脑默认下到C盘的某个不知名的角落占用内存等等。

关于浏览器

  • 谷歌，火狐等浏览器都是不错的选择，但如果你不想下载，事实上win10自带的edge浏览器就是特别特别棒的选择！请一定不要单独下载什么360浏览器，2345浏览器等等。如果自己的主页被篡改了可以再网上搜索一下如何恢复edge默认主页。
  • edge浏览器占用少，速度快，并且干净整洁！！你可以对比一下edge主页和2345主页，很明显清爽很多嘛！！！
  • edge浏览器还有一些好用的功能可以自己去探索一下哦~
